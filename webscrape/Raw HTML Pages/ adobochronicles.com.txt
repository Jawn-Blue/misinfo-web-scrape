window.addEventListener( "message", function( e ) {
	if ( e.origin === "https://r-login.wordpress.com" ) {
		var data = {};
		try {
			data = JSON.parse( e.data );
		} catch( e ) {
			wpcom_remote_login_final_cleanup();
			return;
		}

		if ( data.msg === 'LOGIN' ) {
			// Clean up the login check iframe
			wpcom_remote_login_remove_dom_node_id( "wpcom_remote_login_key" );

			var id_regex = new RegExp( /^[0-9]+$/ );
			var token_regex = new RegExp( /^.*|.*|.*$/ );
			if (
				token_regex.test( data.token )
				&& id_regex.test( data.wpcomid )
			) {
				// We have everything we need to ask for a login
				var script = document.createElement( "script" );
				script.setAttribute( "id", "wpcom_remote_login_validate" );
				script.src = '/remote-login.php?wpcom_remote_login=validate'
					+ '&wpcomid=' + data.wpcomid
					+ '&token=' + encodeURIComponent( data.token )
					+ '&host=' + window.location.protocol
						+ '//' + window.location.hostname;
				document.body.appendChild( script );
			}

			return;
		}

		// Safari ITP, not logged in, so redirect
		if ( data.msg === 'LOGIN-REDIRECT' ) {
			window.location = 'https://wordpress.com/log-in?redirect_to=' + window.location.href;
			return;
		}

		// Safari ITP, storage access failed, remove the request
		if ( data.msg === 'LOGIN-REMOVE' ) {

			document.cookie = 'wordpress_com_login_access=denied; path=/; max-age=31536000';

			return;

			console.log( 'request access: safari' );

				return;

			iframe.setAttribute( 'scrolling', 'no' );

<p><strong>QUEZON CITY, </strong>Philippines (<em>The Adobo Chronicles, </em> Quezon City Bureau) –  A PR photo showing Leni Robredo inspecting one of her rural toilet projects is now the source of bad blood between the Philippine Vice President and Facebook.</p>
<p>It all started when <em>writer/blogger Krizette Laureta Chu posted an opinion piece on her Facebook page in which she talked about Robredo’s project. Her post came with a photo of Robredo during her inspection.</em></p>
<p><em>Alas, Facebook blurred the photo, saying that it may be sensitive to some people.  Facebook did not specify whether the sensitive issue was the toilet bowl in the photo, or of Robredo herself.</em></p>
<p>Immediately, the Office of the Vice President sent a “note verbale” (Robredo has been making the rounds of foreign diplomatic offices lately) to Facebook protesting the giant social media platform’s action.</p>
<p>“How could Facebook decide on its own that the Vice President’s photo is sensitive to some people,” the OVP asked. “She was dressed appropriately in her favorite lavander color, and she was wearing a face mask as required in public,” it added.</p>
<p><em>The Adobo Chronicles</em> tried to get the side of Mark Zuckerberg, but his staff told us he was working from home when we called, and they could not give us his home phone number because it was confidential.</p>
<p>This is a developing story.</p>

<h1 class="entry-title"><a href="https://adobochronicles.com/2021/03/15/leni-robredos-camp-confirms-maria-ressas-war-zone-philippines/" rel="bookmark">Leni Robredo’s Camp Confirms Maria Ressa’s ‘War Zone Philippines’</a></h1>
<div class="entry-meta">
<span class="entry-date"><a href="https://adobochronicles.com/2021/03/15/leni-robredos-camp-confirms-maria-ressas-war-zone-philippines/" rel="bookmark"><time class="entry-date" datetime="2021-03-15T01:49:25-07:00">March 15, 2021</time></a></span> <span class="byline"><span class="author vcard"><a class="url fn n" href="https://adobochronicles.com/author/adobochron/" rel="author">Pol Pinoy</a></span></span> <span class="comments-link"><a href="https://adobochronicles.com/2021/03/15/leni-robredos-camp-confirms-maria-ressas-war-zone-philippines/#respond">Leave a comment</a></span>

<div class="wp-block-image"><figure class="aligncenter size-large"><img alt="" class="wp-image-58813" data-attachment-id="58813" data-comments-opened="1" data-image-description="" data-image-meta='{"aperture":"0","credit":"","camera":"","caption":"","created_timestamp":"0","copyright":"","focal_length":"0","iso":"0","shutter_speed":"0","title":"","orientation":"1"}' data-image-title="3f890e7c-f2db-4889-a089-fd6765a9ae79" data-large-file="https://adobochroniclesdotcom.files.wordpress.com/2021/03/3f890e7c-f2db-4889-a089-fd6765a9ae79.jpeg?w=474" data-medium-file="https://adobochroniclesdotcom.files.wordpress.com/2021/03/3f890e7c-f2db-4889-a089-fd6765a9ae79.jpeg?w=225" data-orig-file="https://adobochroniclesdotcom.files.wordpress.com/2021/03/3f890e7c-f2db-4889-a089-fd6765a9ae79.jpeg" data-orig-size="563,750" data-permalink="https://adobochronicles.com/3f890e7c-f2db-4889-a089-fd6765a9ae79/" sizes="(max-width: 563px) 100vw, 563px" src="https://adobochroniclesdotcom.files.wordpress.com/2021/03/3f890e7c-f2db-4889-a089-fd6765a9ae79.jpeg?w=563" srcset="https://adobochroniclesdotcom.files.wordpress.com/2021/03/3f890e7c-f2db-4889-a089-fd6765a9ae79.jpeg 563w, https://adobochroniclesdotcom.files.wordpress.com/2021/03/3f890e7c-f2db-4889-a089-fd6765a9ae79.jpeg?w=113 113w, https://adobochroniclesdotcom.files.wordpress.com/2021/03/3f890e7c-f2db-4889-a089-fd6765a9ae79.jpeg?w=225 225w"/></figure></div>

<p><strong>MANILA, </strong>Philippines (<em>The Adobo Chronicles, </em>Manila Bureau) – The camp of Vice President Leni Robredo has confirmed what Rappler CEO Maria Ressa has been saying all along — that the Philippines is a “war zone.”</p>
<p><em>“It’s the worst war zone I’ve ever been in,” Ressa likes to say whenever she is interviewed by the Western media.</em></p>
<p><em>The Robredo camp’s confirmation came through what appeared like a 2022 Presidential campaign poster which shows the VP in front of the Philippine flag.</em></p>
<p>But there is one problem.</p>
<p><em>The law and protocols dictate that when hanging or displaying the Philippine flag vertically, the red stripe should be on the right side of the person looking at the flag.  The Robredo poster had the red stripe on the left.</em></p>
<p><em>Every Filipino who has gone through civics lessons in school knows that when the red stripe is on the left, it means the Philippines is at war.</em></p>
<p>Yes, a war zone, as per Ressa.</p>

<p><strong>LOS ANGELES, </strong>California (<em>The Adobo Chronicles, </em>L.A. Bureau) – The Los Angeles Police Department (LAPD) has issued as subpoena to Rodrigo Duterte to appear in California to explain <em>what police found to be the Philippine President’s fake driver license. </em> <em>The California ID was found in an undisclosed location, and it had the President’s photograph.  The license was under the name of a “Norman D.”</em></p>
<p>LAPD told <em>The Adobo Chronicles</em> that it has launched an investigation for possible violation of federal and state laws on falsification of public documents.</p>
<p>“It’s either Duterte stole and faked the license of Norman D, or Norman D faked his license using Duterte’s photo,” LAPD said.</p>
<p>The LAPD acknowledges that it would be very difficult for Duterte to personally appear in California because of current Covid-19 travel restrictions, particularly for senior citizens. <em>Duterte turns 76 on March 28. </em>However, the Department said Duterte can designate an alter ego who is able to travel and personally appear in Los Angeles as soon as possible.</p>
<p>When asked about the subpoena, Presidential Spokesperson Harry Roque said, “Hindi ko po alam.  Wala pong nabanggit ang Pangulo sa akin.” (I don’t know. The President has not mentioned anything to me.)</p>
<p>This is a developing story.</p>
